# 첫 번째 과제 - 게시글 만들기



### 기능 사항 개요

- 게시글
    - 사용자는 게시글을 생성, 수정 및 삭제가 가능합니다.
    - 사용자는 모든 게시글을 조회할 수 있습니다.



### 요구 사항 구체

- 게시글
    - 게시글은 다음으로 구성 됩니다.
        - 작성자 아이디 (문자열)
        - 내용 (문자열)
        - 생성날짜 (타임스탬프)
        - 수정날짜 (타임스탬프)
